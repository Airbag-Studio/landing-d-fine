# D-Fine — Landing Page

Landing page for **D-Fine**, an iOS + Apple Watch app that tracks vitamin D synthesis from sunlight, food, and supplements.

Built by [Airbag Studio Srl](https://airbag.studio).

---

## Stack

- Static HTML
- Tailwind CSS via CDN (no build step)
- Vanilla JS (scroll animations only)

No frameworks. No dependencies. Deployable anywhere.

---

## Development

Open `index.html` directly in a browser, or serve locally:

```bash
npx serve .
# or
python3 -m http.server 8080
```

---

## Deploy

Compatible with GitHub Pages, Netlify, Vercel, or any static host. No build step required.
